# Mobile-Aware Smart Contracts Lifecycle

## Overview

This repository contains the implementation, formal models, experiments,
and supporting artefacts for the research project:

"A Sharding-based Formalized Lifecycle for Scalable Mobile Smart Contracts"

## Repository Structure

- cpn-model/
- goal-models/
- security-analysis/
- experiments/
- results/

## Requirements

- CPN Tools 4.0.1
- Java 8
- Windows 11

## Features

- Hierarchical Colored Petri Net model
- Mobile-aware shard routing
- Cross-shard verification
- ISSRM-based security analysis
- Scalability experiments
